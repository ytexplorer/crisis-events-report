# docs/solutions

One learning per file. When something costs you more than ten minutes, the fix
goes here so no future session pays for it twice.

Convention:

- Filename: `YYYY-MM-DD-short-slug.md`
- YAML frontmatter with `date`, `tags` and a one-line `problem`
- Body: symptom, cause, fix. Terse. A future agent greps this directory
  before it starts debugging.

See `2026-07-06-example-follow-redirects.md` for the shape. Delete it once you
have a real one.
